# snake-water-gun-game
A small Python game made during my Code With Harry course.

This is a simple text-based game built using Python. I created this project while learning from the "Code With Harry" Python course.

## How to Play
1. Run the Python file in your terminal.
2. Choose **S** for Snake, **W** for Water, or **G** for Gun.
3. The computer will make its choice randomly, and the winner will be decided!
